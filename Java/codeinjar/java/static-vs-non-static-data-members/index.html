Static and Non static data members in java

We know that Java class members are fields and methods. A field or data member can be static or no-static. If a class member like field is static then we don't need to create object of the class to access it.

See an example below


    public class Main {
 
//static data member 
static String name = "Mark Smith";
static int age = 30;

   public static void main(String[] args) {
    
//call the printSummation () method inside main method
System.out.println("Name is "+ name);
System.out.println("Age is "+ age);

   }

}
/**
*output:
* Name is Mark Smith
* Age is 30
* 
*/

You see, we don't need to create object of the class to access the name and age data members as they are static.

But, if they are not static, then you probably have to use create method to access them, otherwise you will get error.

See another example, where we have created an object to access the non-sratic data members. 


   public class Main {
   
   //static data member
   String name = "Mark Smith";
   int age = 30;
   
   public static void main(String[] args) {
   
   //creating an object of Main class
   Main obj = new Main();
   
   //print name and age using the obj of the class
   System.out.println("Name is "+ obj.name);
   System.out.println("Age is "+ obj.age);
   
   }
   
   }
   /**
   *output:
   * Name is Mark Smith
   * Age is 30
   *
   */

The same logic will also be applied other types of variables or methods of the class.
